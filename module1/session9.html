Session Objectives

By actively engaging with the specified Coursera content for this session, you will aim to:

  • Reinforce your understanding of core Big Data concepts.
  • Gain a foundational understanding of what Apache Hadoop is and its primary purpose in the Big Data ecosystem.
  • Become familiar with the basic concepts of the Hadoop Distributed File System (HDFS).
  • Get an initial introduction to the MapReduce programming model.
  • Identify other key components within the broader Hadoop ecosystem (e.g., Hive, Pig at an introductory level).

Introduction to this Blended Learning Session

This session is dedicated to self-paced learning using the highly recommended Coursera course: "Hadoop Platform and Application Framework." The 3 hours allocated are for you to dive into the initial modules of this course. Blended learning combines traditional classroom instruction with online educational materials and opportunities for interaction online. This approach allows you to learn foundational concepts at your own pace, which we will then build upon in subsequent in-class sessions and labs.

Effective engagement with the Coursera content is crucial for building a strong understanding of Hadoop, which is a cornerstone technology in Big Data.

Focus Areas for Session 9 (Approx. first 3 hours of Coursera content)

For this 3-hour blended learning session, aim to cover approximately the first week's content (and potentially the beginning of the second week's content) of the Coursera course. This typically includes:

  • Introduction to Big Data (Recap & Deeper Dive):
    • Revisiting the definition and significance of Big Data.
    • Further exploration of the 5 V's (Volume, Velocity, Variety, Veracity, Value) with examples.
    • Understanding the challenges Big Data poses to traditional systems.
  • Introduction to Hadoop:
    • What is Hadoop? Its history and motivation.
    • The Hadoop Ecosystem: A high-level overview of its various components.
    • Core Hadoop Components: HDFS and MapReduce (introduction).
    • Common Hadoop use cases.
  • Hadoop Distributed File System (HDFS) - Fundamentals:
    • HDFS architecture (NameNode, DataNode concepts).
    • Data blocks and replication for fault tolerance.
    • How files are read from and written to HDFS (conceptual).
  • MapReduce - Introduction (if covered in early Coursera modules):
    • Basic concept of the MapReduce programming paradigm.
    • The Map function and the Reduce function.
    • A simple conceptual example (e.g., word count).

Note: The exact module breakdown on Coursera can change. Adapt your study to cover these core introductory topics within the allocated 3 hours.

Suggested Learning Activities

To make the most of this blended learning session:

  • Watch all lecture videos provided in the relevant Coursera modules.
  • Read any supplementary materials or articles linked within the course.
  • Take notes on key concepts, definitions, and architectures.
  • Actively participate in any quizzes or short assessments provided by Coursera to check your understanding.
  • If there are simple hands-on exercises or demos in these initial modules, try to follow along.
  • Reflect on how these concepts relate to what you've learned in Sessions 1-5 of this course.
  • Jot down any questions you have for discussion in future class sessions.

Key Takeaways for Session 9

  • Self-paced blended learning is an effective way to cover foundational material in depth.
  • A strong grasp of Big Data fundamentals and an introduction to Hadoop's purpose and core components (HDFS, MapReduce concept) are key outcomes for this session.
  • The Coursera course provides valuable supplementary learning to the in-class material.
  • This session prepares you for more detailed discussions on Hadoop architecture and practical labs in later modules.
