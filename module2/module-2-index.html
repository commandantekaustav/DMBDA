Module II Focus: Data Preprocessing & Core Mining Techniques

This module dives into the essential steps of preparing data for analysis and introduces fundamental data mining algorithms. You will learn various techniques for data cleaning, transformation, and normalization. We will then explore core mining tasks including clustering for finding natural groupings, classification for predicting categories, and association rule mining for discovering relationships in data. Practical labs will focus on implementing these techniques using Python and relevant libraries. This module also includes the second phase of your mini-project and further blended learning on Big Data platforms.
