2. Algorithmic Bias

What it is: Algorithmic bias occurs when a computer system reflects the implicit values and biases of the humans who created it or the data it was trained on. Since AI models learn from data, if the data reflects existing societal biases, the model will learn and often amplify those biases.

Key Data Privacy Regulations

In response to these challenges, governments around the world have enacted data privacy laws. Understanding their principles is now a required skill for data professionals.

GDPR (Europe)

The General Data Protection Regulation is a comprehensive data protection law for the EU. Key principles include:

  • Lawful Basis for Processing
  • Data Minimization
  • Right to Access & Erasure

CCPA / CPRA (California)

The California Consumer Privacy Act gives consumers more control over their personal information. Key principles include:

  • Right to Know
  • Right to Delete
  • Right to Opt-Out of Sale

DPDP Act (India)

India's Digital Personal Data Protection Act, 2023, establishes a framework for processing digital personal data. Key roles are:

  • Data Fiduciary: The entity that determines the purpose and means of processing data (the company).
  • Data Principal: The individual to whom the personal data relates (the user).
  • Core Principle: Personal data can only be processed for a lawful purpose with the individual's consent.
